# How to run

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

### `npm install`

앱 실행에 필요한 패키지들을 설치합니다.

### `npm start`

앱을 실행합니다. [http://localhost:3000](http://localhost:3000)

### `npm test`

아래 테스트 코드들에 대한 테스트를 진행합니다.

- resource_presenter.test.js
- ResourceItem.test.js
- App.test.js
